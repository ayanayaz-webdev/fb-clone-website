# FB Clone Website

This is a Facebook website clone that I built while learning Git and GitHub. The main purpose of this project was to practice version control, understand Git workflows, and become familiar with using GitHub to manage and share projects.

Although the project was created as part of a Git tutorial, it also helped me improve my frontend development skills by recreating the basic layout and design of Facebook using HTML and CSS.

## Features

- Facebook-inspired homepage layout
- Responsive and clean user interface
- Navigation bar
- Sidebar sections
- News feed design
- Profile and post styling
- Organized project structure

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub

## What I Learned

While building this project, I learned how to:

- Initialize a Git repository
- Track and commit changes
- Work with branches
- Push projects to GitHub
- Manage project versions
- Build a structured frontend layout

## Purpose

This project was created for educational purposes to strengthen both my frontend development and Git/GitHub skills. It is not intended to be a fully functional clone of Facebook but rather a practice project demonstrating my understanding of web development fundamentals and version control.
